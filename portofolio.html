Tentang Saya

Saya Farhan Ramadhan, berasal dari Jakarta, dan saat ini sedang menempuh pendidikan Sarjana di Universitas Pancasila. Sebagai mahasiswa Teknik Informatika angkatan 2022, saya memiliki spesialisasi dalam pengembangan model machine learning dan analisis data. Dengan kombinasi kemampuan teknis dalam Python, framework machine learning seperti tensorflow, dan tools data analysis, saya berkomitmen untuk menciptakan solusi berbasis data yang berdampak nyata.
